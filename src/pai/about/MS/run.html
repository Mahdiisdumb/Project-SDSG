About Mahdi Studios

Mahdi Studios — led by Mahdiisdumb — creates approachable games and open-source tools for learners and hobbyists.

Overview

Mahdi Studios, maintained by Mahdiisdumb, focuses on lightweight HTML and Unity projects that run across a wide range of systems (Windows, Linux, and Unix-like platforms). Projects are provided as open-source packages so users can run them locally and inspect the source.

Notable projects

  • Martani 360
  • Project SDSG (School Defying Software Games)
  • Various experimental and discontinued titles (archived)
  • Dramitic Adhan
